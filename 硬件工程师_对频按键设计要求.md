# SiK 数传模块 — 新增"对频按键"硬件设计要求

## 一、需求简述

在现有 HM-TRP 兼容数传模块的基础上，**新增一个独立的物理按键**，用于触发"一键对频"功能（类似 ELRS 接收机的 Bind 按键）。用户长按此按键 3 秒，模块进入对频模式，自动与另一端配对。

此按键与现有的 CONFIG 引脚（P0.2，用于进入 Bootloader）**完全独立，互不干扰**。

---

## 二、引脚分配

| 项目 | 值 |
|:---|:---|
| MCU 引脚 | **P1.7 (Pin 8)** |
| 有效电平 | **低电平有效**（按下接地） |
| 引脚模式 | 数字输入，开漏模式，内部弱上拉 |

### 为什么选 P1.7？（经数据手册验证）

根据 Si1000 数据手册 **Table 3.1 Pin Definitions** 和 **Figure 3.1 Pinout Diagram**，在 42-pin LGA 封装中，**P1.0~P1.4 没有引出到芯片外部**，它们是芯片内部 SPI1 总线（连接 MCU 与内置 EZRadioPRO 射频引擎），无法连接任何外部电路。

P1 端口实际引出到封装外的引脚只有 3 个：

| 引脚 | Pin 号 | 当前用途 | 是否可用 |
|:---:|:---:|:---|:---:|
| P1.5 | Pin 10 | LED_GREEN — 绿色指示灯 | 已占用 |
| P1.6 | Pin 9 | LED_RED — 红色指示灯 | 已占用 |
| **P1.7** | **Pin 8** | **未使用，空闲 GPIO** | **可用** |

**P1.7 是 P1 端口上唯一一个既引出到封装外、又没有被现有设计占用的引脚。**

> **注意**：
> - P1.0~P1.4 是芯片内部 SPI 总线，**没有外部焊盘**，绝对不能使用
> - P1.5 和 P1.6 已被 LED 占用
> - 如果 P1.7 在你的板子上已有其他用途（如 PA 控制），请联系软件工程师评估改用 P2 端口或 P0.6 等其他空闲引脚

## 三、电路设计

### 3.1 原理图

```
        VDD_MCU (3.3V)
            │
           [R1]  10kΩ  上拉电阻
            │
            ├──────────── P1.7 / Pin 8 (Si1000 MCU)
            │
           [C1]  100nF (104) 陶瓷电容
            │
           [SW1] 轻触开关 (按下导通)
            │
           GND
```

### 3.2 元器件清单

| 编号 | 元件 | 规格 | 封装建议 | 用途 | 是否必须 |
|:---:|:---|:---|:---|:---|:---:|
| SW1 | 轻触开关 | 常开，额定 50mA/12V | 3×6mm / 4×4mm 贴片 或 6×6mm 直插 | 对频触发按键 | **必须** |
| R1 | 上拉电阻 | 10kΩ ±5% | 0402 / 0603 | 确保未按下时引脚稳定为高电平 | **强烈建议** |
| C1 | 去抖电容 | 100nF (104) 陶瓷 | 0402 / 0603 | 硬件消抖，滤除按键抖动毛刺 | **强烈建议** |

### 3.3 各元件说明

**SW1 — 轻触开关**
- 一端接 P1.7 (Pin 8)，另一端接 GND
- 按下时 P1.7 被拉低到 GND（低电平有效）
- 松开时 P1.7 通过 R1 上拉回 VDD（高电平）
- 按键选型无特殊要求，普通轻触开关即可

**R1 — 10kΩ 上拉电阻**
- 虽然 Si1000 内部有弱上拉（约 100kΩ 级别），但在射频模块的高电磁干扰环境下，内部弱上拉**极易被干扰导致误触发**
- 外部 10kΩ 上拉可将抗干扰能力提升约 10 倍
- 如果空间实在紧张可以省略，但不推荐

**C1 — 100nF 去抖电容**
- 并联在 P1.7 与 GND 之间（与按键并联）
- 作用：按键闭合瞬间的机械抖动（通常持续 5~20ms）会被电容滤波吸收
- 有了这个电容，软件端只需做极简的去抖处理，节省 MCU 运算资源
- 不要使用过大的电容（如 1uF），否则会使按键响应变慢

## 四、PCB 布局建议

1. **按键位置**：放在板子边缘、用户手指容易触及的位置。如果模块有外壳，需要在外壳上开孔或预留凸起
2. **走线**：P1.7 (Pin 8) 的走线尽量短，远离射频天线走线和功率放大器（PA）区域，减少射频耦合导致的误触发
3. **接地**：R1 和 C1 的 GND 引脚就近连接到数字地，不要接到射频地
4. **Pin 8 位置参考**：从 Si1000 Pinout Diagram (Top View) 可知，Pin 8 位于芯片左侧，紧邻 P1.6 (Pin 9) 和 P2.0 (Pin 7)

---

## 五、与现有电路的关系

新增按键电路与以下现有引脚**完全独立**，不需要修改任何现有电路：

| 现有引脚 | 功能 | 是否受影响 |
|:---|:---|:---:|
| P0.2 (CONFIG) | Bootloader 按键 / CTS 流控 | 不影响 |
| P0.3 (ENABLE) | RTS 流控 | 不影响 |
| P0.4 (TX) | 串口发送 | 不影响 |
| P0.5 (RX) | 串口接收 | 不影响 |
| P1.5 (LED_GREEN) | 绿色指示灯 | 不影响 |
| P1.6 (LED_RED) | 红色指示灯 | 不影响 |

---

## 六、验证方法

硬件打样后，可以用以下方法快速验证按键电路是否正常：

1. **万用表测量**：未按下时 P1.7 (Pin 8) 对 GND 电压应为 **~3.3V**；按下时应为 **~0V**
2. **示波器观察**：按下按键时，P1.7 波形应该干净地从高跳到低，无明显抖动（如果有 C1 的话）
3. **软件验证**：待固件完成后，进入 AT 指令模式，可通过指令读取 P1.7 引脚状态确认
